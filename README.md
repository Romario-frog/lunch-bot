# Team Lunch Scheduler Bot

Telegram-бот для зручного планування обідніх перерв команди.

## Що вміє бот

- запланувати обід;
- перенести обід;
- скасувати обід;
- показати мій обід;
- показати, хто зараз на обіді;
- показати розклад обідів за день;
- нагадати про обід за 15 хвилин;
- приймати заявки на доступ;
- розділяти користувачів на операторів та адміністраторів.

## Як оператор користується ботом

1. Відкриває бота і натискає `/start`.
2. Обирає **🍽 Запланувати обід**.
3. Вводить час вручну, наприклад `17:15`.
4. Обирає тривалість: **15 хв**, **30 хв** або **45 хв**.
5. Бот записує обід у таблицю і ставить позначки в розкладі.

## Як працює таблиця

Бот працює з листом `schedule`.

- ФІО операторів — у колонці A.
- Час — у рядку 2.
- Для зайнятого обіду бот ставить позначку **О**.
- Для 15 хв ставиться 1 позначка.
- Для 30 хв ставляться 2 позначки.
- Для 45 хв ставляться 3 позначки.

Бот враховує тільки реальні позначки **О / 0** і не рахує службові рядки типу `ЛІМІТ НА ВИХІД`, `#REF!`, правила тощо.

## Ролі

### Оператор

- може запланувати свій обід;
- може перенести свій обід;
- може скасувати свій обід;
- може переглянути свій обід та поточний розклад.

### Адміністратор

- бачить користувачів;
- підтверджує заявки;
- має розширене меню;
- контролює розклад і логи.

## Важливо

ПІБ у листі `users` має збігатися з ПІБ у `schedule`, але бот також пробує знайти людину за латиницею: наприклад `Latin Name → Прізвище Ім'я`.
